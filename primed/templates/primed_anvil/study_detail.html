Workspaces on AnVIL

The following dropdowns show PRIMED data storage workspaces that exist on AnVIL that are associated with this study. Once data preparation for a workspace is complete, the workspace is shared with the consortium. While data in a workspace is being prepared, the workspace will appear in the tables below but only users who are actively preparing data can access it on AnVIL.

To access a workspace on AnVIL, investigators must:

  1. link their AnVIL account in this app, and
  2. have approval via the correct PRIMED data access mechanism for data in the workspace.

These workspaces contain data from a dbGaP accession. Investigators must be covered under an approved PRIMED coordinated dbGaP application to access these workspaces, even if they have been shared with PRIMED.
